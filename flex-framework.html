Flexgrid Layout Framework

The .flexgrid class functions as a container for flex items. It also plays two roles: .container and .row classes as used in Bootstrap. You can also nest containers. To do so, use .flexgrid-nm which removes the margin used for a parent container i.e., (wrapper for main-content and sidebars)

Our grid system has five default breakpoints based on a twelve columns:

  • Extra extra large (col-xxl-)
  • Extra large (col-xl-)
  • Large (col-l-)
  • Medium (col-md-)
  • Small (col-sm-)

Breakpoints, gutters, margins are all customizable in the variables scss file.

Adding a number between 1-12 tells the class how many columns to span:

  • (col-xxl-9) nine columns
  • (col-l-6) six columns
  • (col-md-2) one column

We also have a class which will shift right a flex item based on the number of grids added to the classname:

  • (col-xxl-8 col-xxl-shift-4) span eight columns, shift four columns
  • (col-xxl-6 col-xxl-shift-6) span six columns, shift six columns
  • (col-xxl-3 col-xxl-shift-9) span three columns, shift nine column
