Grid Layout Framework

For use with repeated column, row content with similar widths. (CTAs, Links with Icons...etc)

The grid-layout() mixin takes a column count, row gap and column gap as variables. See usage below:

.grid-container { @include grid-layout(12, 1rem, 2rem); }

The $row-gap will automagically be the same as $col-gap if you leave it empty.

.grid-item { @include grid-col-start(2); @include grid-col-span(4); @include grid-row-start(1); }
